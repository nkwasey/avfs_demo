Documentation

This API provides weather and fire data for countries within West-Africa and parts of Central Africa. Get access active fire products from MODIS and VIIRS as well as current, daily and 5-day weather forecast information. Please remember that all Examples of API calls that listed on this page are just samples and do not have any connection to the real API service!

Available Fire Data

  • MODIS C6 is available
  • VIIRS 375 m (S-NPP)
