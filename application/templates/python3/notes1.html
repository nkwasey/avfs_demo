Installing Python3 in Windows

  • Created : Oct, 26, 2020
  • Last Updated: Oct, 26, 2020

This documentation is always evolving. If you've not been here for a while, perhaps check again from time to time.

Procedure

  • Visit https://www.python.org/downloads/. Clicking "Download Python 3.9.0", downloads the latest version

  • Alternatively, you can scroll down this page to check out and download previous version of python 3.

  • Double click the downloaded file to commence the installation procedure

    • 1. Select the options as shown in the picture below

    • 2. Check all option under “Optional Features”

    • 3. Check “Install for all users” option in Advanced Options, then hit Install”

    • 4. After successful installation, you should see an IDLE shortcut in the ‘Recently added’ menu
